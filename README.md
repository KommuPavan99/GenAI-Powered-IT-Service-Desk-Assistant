<h1 align="center">🤖 GenAI-Powered IT Service Desk Assistant</h1>

<p align="center">
A modern AI-powered IT Service Desk web application developed using <b>Django</b> and a <b>Local Large Language Model (Microsoft Phi-3 Mini)</b> running through <b>Ollama</b>. The system automates incident management, provides intelligent troubleshooting assistance, and enables organizations to resolve common IT issues efficiently while keeping all AI processing completely offline for enhanced privacy and security.
</p>

<hr>

<h2>📌 Introduction</h2>

<p>
The <b>GenAI-Powered IT Service Desk Assistant</b> is an intelligent web application designed to simplify IT support operations by integrating traditional IT Service Management (ITSM) practices with Generative Artificial Intelligence. The application enables users to report incidents, search a centralized Knowledge Base, and receive AI-generated troubleshooting recommendations through a locally hosted Large Language Model.
</p>

<p>
Unlike cloud-based AI solutions, this project utilizes <b>Microsoft Phi-3 Mini</b> running locally with <b>Ollama</b>, ensuring complete data privacy, offline accessibility, and zero dependency on external AI APIs. The system first searches its Knowledge Base for existing solutions and, when necessary, generates contextual responses using the local AI model. This hybrid approach improves issue resolution while maintaining the reliability of documented organizational knowledge.
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
The application serves as a centralized platform for managing IT support requests. It provides users with an intuitive interface for creating incidents, browsing troubleshooting documentation, searching technical articles, and interacting with an AI-powered assistant. Administrators can manage incidents and Knowledge Base articles through the Django Administration Dashboard, while end users benefit from intelligent recommendations generated by the local AI model.
</p>

<p>
The project demonstrates the practical integration of Artificial Intelligence into IT Service Management, combining rule-based information retrieval with modern Large Language Models to improve user experience and reduce support resolution time.
</p>

<hr>

<h2>✨ Key Features</h2>

<ul>
<li>🏠 Modern and responsive Home Page for seamless navigation.</li>

<li>📊 Interactive Dashboard displaying incident statistics, system overview, and quick access to major modules.</li>

<li>📝 Incident Management module for creating, tracking, and managing IT support requests.</li>

<li>📚 Knowledge Base containing predefined troubleshooting articles for frequently occurring IT issues.</li>

<li>🔍 Intelligent Knowledge Base Search using keyword-based filtering for quick information retrieval.</li>

<li>🤖 AI Assistant capable of understanding natural language queries and generating troubleshooting recommendations.</li>

<li>🧠 Local Large Language Model integration using Ollama and Microsoft Phi-3 Mini without relying on cloud services.</li>

<li>⚡ Automatic incident categorization based on the user's problem description.</li>

<li>🔥 Automatic priority prediction to help prioritize critical IT incidents.</li>

<li>👨‍💼 Django Administration Dashboard for secure management of incidents and Knowledge Base articles.</li>

<li>📱 Responsive Bootstrap-based user interface compatible with desktop and mobile devices.</li>

<li>🔒 Offline AI processing ensuring complete privacy, security, and reduced operational cost.</li>
</ul>

<hr>

<h2>🛠️ Technologies Used</h2>

<p>
The project integrates multiple modern web development and Artificial Intelligence technologies to build a scalable and intelligent IT Service Desk application.
</p>

<ul>
<li><b>Python</b> – Core programming language used for backend development.</li>

<li><b>Django</b> – High-level Python web framework for application development.</li>

<li><b>SQLite</b> – Lightweight relational database for storing incidents and Knowledge Base articles.</li>

<li><b>HTML5</b> – Structure of web pages.</li>

<li><b>CSS3</b> – Styling and layout customization.</li>

<li><b>Bootstrap 5</b> – Responsive UI framework.</li>

<li><b>JavaScript</b> – Client-side interactivity.</li>

<li><b>Ollama</b> – Local AI runtime used for executing Large Language Models.</li>

<li><b>Microsoft Phi-3 Mini</b> – Lightweight Local Large Language Model for AI-powered troubleshooting.</li>

<li><b>Git & GitHub</b> – Version control and project hosting.</li>
</ul>

<hr>

<h2>🤖 Local AI Setup</h2>

<p>
The AI Assistant operates entirely on the local machine using Ollama and Microsoft's Phi-3 Mini model. Unlike cloud-based AI services, no user data leaves the system, ensuring privacy and offline functionality.
</p>

<pre>
ollama pull phi3:mini

ollama serve
</pre>
