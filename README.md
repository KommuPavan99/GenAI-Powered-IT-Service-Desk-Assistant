<h1 align="center">🤖 GenAI-Powered IT Service Desk Assistant</h1>

<p align="center">
An AI-powered IT Service Desk web application developed using <b>Django</b> and a <b>Local Large Language Model (Phi-3 Mini)</b> through <b>Ollama</b> to automate IT support, incident management, and troubleshooting.
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
The <b>GenAI-Powered IT Service Desk Assistant</b> is an intelligent web-based application that simplifies IT support by combining a traditional Knowledge Base with Generative AI.

Instead of relying only on predefined troubleshooting articles, the system first searches its Knowledge Base and then consults a locally hosted AI model (Phi-3 Mini via Ollama) to generate intelligent troubleshooting recommendations.

Since the AI model runs locally, the application works without cloud APIs, ensuring better privacy, reduced operational costs, and offline accessibility.
</p>

<hr>
